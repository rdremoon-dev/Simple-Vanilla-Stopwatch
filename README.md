# ⏱️ Simple Vanilla Stopwatch

A functional web-based stopwatch built to practice JavaScript logic and DOM manipulation. This project focuses on the core mechanics of time tracking.

## 🔗 Live Demo
View the project here: [https://stopky-012.netlify.app/](https://stopky-012.netlify.app/)

## 🎯 Project Purpose
The goal of this project was to master:
*   Using `setInterval` and `clearInterval` in JavaScript.
*   Updating the UI dynamically as time progresses.
*   Managing state (Start, Pause, Reset) within a web application.

## ✨ Current Features
*   **Start:** Begins the timer with millisecond precision.
*   **Pause:** Freezes the current time.
*   **Reset:** Clears the timer and returns the display to zero.

## 🛠️ Technologies Used
*   **HTML5:** Project structure.
*   **CSS3:** Custom layout and styling.
*   **JavaScript:** Core timing logic.
